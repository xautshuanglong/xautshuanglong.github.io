| 快捷键组合 | 操作效果 |
|:----------:|:--------:|
| Ctrl + a | 切换到命令行开始 |
| Ctrl + e | 切换到命令行末尾 |
| Ctrl + l | 清除屏幕内容，同 Clear。 |
| Ctrl + u | 剪切清除光标之前的内容 |
| Ctrl + k | 剪切清除光标之后的内容 |
| Ctrl + y | 粘贴所剪切的内容 |
| Ctrl + r | 在历史命令中查找 |
| Ctrl + c | 终止命令 |
| Ctrl + d | 退出 shell，logout |
| Ctrl + z | 转入后台进行，直到用户退出终止。`&`不会终止 |
| !! | 重复执行最后一条命令 |
| history | 显示历史命令，`！`执行某条历史命令 |
| Ctrl + p `↑` | 显示上一条命令 |
| Ctrl + n `↓` | 显示下一条命令 |
| !$ | 显示系统最近的一条参数 |
